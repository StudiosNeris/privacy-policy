Data Collection and Usage

  • Personal Data: The application does not directly collect or store personally identifiable information (such as name, email, or phone number) from its users.
  • Game Data: Game progress and settings are stored locally on the user's device.
  • Firebase Analytics and Authentication: We utilize Google Firebase services for application operation:
    • Authentication (Anonymous): The app signs in users anonymously using Firebase Auth. This generates a unique, non-personal User ID (UID) linked to the device, which is used internally by Firebase to distinguish users and is not personally identifiable.
    • Remote Configuration: We use Firebase Remote Config to dynamically update app settings (like the selected mini-game type) without requiring an app update. This process does not collect personal data.
    • Firebase services may automatically collect certain device and usage data (e.g., crash reports, session length, device model) as described in their documentation.
  • Advertising (AdMob): The application displays third-party advertisements via Google AdMob.
    • AdMob uses identifiers like your device's advertising ID to serve personalized ads and measure ad effectiveness.
