1. Data Collected and Purpose of Use

Our Apps use indirect data collection methods for the following purposes:

a. Application Usage Analysis (Firebase Analytics)

We use **Firebase Analytics**, a service provided by Google, to measure App performance and improve the user experience. The collected data includes:

  • **App Activity:** Application interactions and usage patterns.
  • **Device Information:** Device Identifiers (such as Firebase installation ID), Crash logs, and diagnostics data.

This data is used for **App functionality** and **Analytics** purposes.

b. Advertising Services (Google AdMob)

Google AdMob advertisements are displayed within the App. This advertising service may collect data to serve ads relevant to user interests. The collection, use, and protection of such data are entirely subject to Google's Privacy Policy.

2. Data Storage and Sharing

  • **Local Data:** The App stores game progression data **only locally on the user's own device**.
  • **Shared Data:** Collected Analytics data and Advertising data are shared with Google (Firebase and AdMob) for analysis and service delivery purposes.
  • **Encryption:** All collected data is encrypted during transmission from the device to the servers.
